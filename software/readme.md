## Development:

Add "billardtrainer/software" to system PATH (environment variables)  
Usage (working folder = `/billardtrainer/software/`):  
`(python|python3) main.py [-h]`  

Optional arguments:  
  ```
  -h, --help            show this help message and exit
  ```  
Example run configurations:  
```  
python3 main.py -h
```
